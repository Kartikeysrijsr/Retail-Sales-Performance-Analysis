# Retail Sales Performance Analysis 📊

## Project Overview
This project is an end-to-end Data Analytics pipeline designed to process, analyze, and visualize a large-scale online retail dataset (500,000+ records). It transforms raw transactional data into actionable business intelligence for executive decision-making.

## Technology Stack 🛠️
* **Data Cleansing & EDA:** Python (Pandas, Matplotlib, Seaborn)
* **Database & Aggregation:** MySQL (Star Schema, Views)
* **Data Visualization:** Microsoft Power BI (DAX, Interactive Dashboards)

## Key Achievements 🏆
* Cleaned and pre-processed over half a million rows, removing 135k+ anomalous records using Python algorithms.
* Optimized data retrieval by 90% using pre-aggregated SQL Views.
* Developed an interactive dashboard highlighting that the UK market accounts for ~80% of total revenue.

## Files Included 📁
* `Retail_Data_EDA.ipynb`: Python code for data cleaning and exploration.
* `Database_Schema.sql`: SQL scripts for table creation and data aggregation.
* `Retail_Sales_Dashboard.pbix`: The final interactive Power BI file.
* `Project_Report.pdf`: Comprehensive documentation of the project.
